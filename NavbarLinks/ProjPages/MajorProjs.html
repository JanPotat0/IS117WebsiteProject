Python Pandas

In my "Building Web Applications" class, my professor assigned a project where we would have to work with a data set - state and population. For the requirements, the project would be grabbing data of the given csv file. It should get the min and max, calculate the standard deviation, calcuate a sample size of a given confidence interval and margin of errors, and print the results to a csv file in a desginated folder using pandas.

The code shown on the right takes in a .csv file - state_population.csv, reads and puts it into a pandas data frame for calculations. As mentioned before, the program is supposed to get the min and max, the length or row count, and the calculate the standard deviation. Separately from these calculations, the program had to calculate a sample size of varying confidence intervals for each margin of error value. In the third screenshot, a nested for loop is used to iterate through each margin of error with iterating confidence intervals - thus producing fifteen .csv files named as such - sample_data_number_confidence interval_margin of error_line count.
